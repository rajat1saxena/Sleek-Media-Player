The interface of the player is self explanatory

Running the player
-----------------------------------------------------
cd to the sleek player directory and then run "guibuild.py"

Features
---------------------------------------------------------
1. The player is based on GStreamer
2. It can play all the files which gstreamer can play
3. It supports seek,navigation and basic play,pause and stop features
4. It supports shoutcast playlist(".pls").So you can play your totem playlists with this player


Some points to note
-----------------------------------
1.) If some audio file is playing and you want to play another file,in that case you have to stop the currently playing file then select new file and press play
